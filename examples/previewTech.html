The Products

What we do

We're building state of the art web applications which offer interactive and responsive user experiences to millions of customers. Our technology stack is tailored to serve advanced applications at the highest speeds.

We focus on social networking sites, which allow our users to connect, interact and communicate in real-time. We believe that modern web application are becoming the most powerful way of delivering this communicative functionality.

Our framework powers our flagship product; a website has 60 million pageviews every month and:

  • 7000 chat messages / hour
  • 2000 new friendships / hour
  • 300 created user profiles / hour

To achieve good speed performance we run and maintain our services on dedicated machines. We ♡ our squeeze boxes.

CM Framework

What we work with

Our in-house web application framework written with PHP5 allows us to achieve the flexibility we need for our line of products. During its long development we focused on modularity, extendibility and reusability.

The fully object-oriented design and the clean structure make the framework easy to grasp and work with. We continuously put effort into high quality code and we automate everything we need to do more than once, especially tests, for maximal efficiency.

To scale up to our applications' demands we combine powerful and proven open source technologies like Elasticsearch, Redis, Memcache or Node.js. These and various other services are built-in and readily accessible by our backend APIs.

The framework provides tools to develop front-end application code which delivers a remarkable user experience. Ranging from HTML and CSS templates to sophisticated JavaScript models we use the best available tools and practices. We deeply integrated Backbone.js, Lessphp and Smarty to allow for rapid development of rich user interfaces.
